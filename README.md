# Task Scheduler


Aplicação para agendamento de tarefas.


> Frontend: *[ReactJS](https://pt-br.reactjs.org/)*. <br>
> Backend: *[NodeJS](https://nodejs.org/en/)*. <br>
> Banco: KnexJS - *[MySQL](http://knexjs.org/#Installation-node)*. <br>


Backend:

```
cd backend

npm install

npx knex migrate:latest

npm start
```

---

Frontend:

```
cd frontend

npm install

npm start
```
